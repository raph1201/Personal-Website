COURSES

SOFTWARE DESIGN

Programming course in Java. Extensive introduction to state-of-the-art software engineering, with emphasis on Encapsulation, Classes and Interfaces, UML Diagrams, Test suites and branch coverage.

LANG & PARADIGMS

Programming course in OCamL. In-depth look into the benefits and strength of functionnal programming. Implementation of variable checking, substitution, expression evaluation & unification.

STATS & PROBABILITIES

Discrete and Continuous random variable analysis. Parameter Estimation using Method of Moments and Maximum Likelihood. Bias, Consistency, Error estimation and comparison of estimators. Hypothesis testing.

OPERATING SYSTEMS

Full course about OS infrastructure, features and role in a computer. Processes & Threads synchronisation, parallelization, and communication . Virtual & Real time Scheduling algorithms. Memory management and File system.

DISCRETE MATHEMATICS

Problem-solving course diving deep in the applications and beauty of discrete mathematical problems, and relation to real-life problems. Included Counting Theory, Graph Theory, Pigeonhole Principle, Combinatorics, Recurrences, Genereting Fucntions and Discrete Probabilities.

NON-LINEAR DYNAMICS

Study of non-linear dynamical systems, including their existence, uniqueness, fixed points and bifurcations. Use of vector fields and phase portraits representations. Linearization using Jacobian matrices. Study of collective dynamics and Chaos.

ARTIFICIAL INTELLIGENCE

Introduction to the intricacies of modern AI. Focus on search methods, problem-solving, adapting to uncertainties, and application to Game Theory. Study of Reinforcement Learning methods, such as Minimax and Monte Carlo Tree Search. Policy design.

CLOUD COMPUTING

Large overview of the Cloud industry, from its architecture to its economy. Study of Cloud Service models, Networking, Virtualization, Scheduling, Load Balancing, Ressource Management, Data Streaming and Security. Comparison of past cloud computing projects and models.

DATA SCIENCE

Introduction to Data Science and Data Analysis with JSON. Study of Data Mining and Data Visualization using Python librairies like BS4 to fetch informations and Bokeh to display them for analysis. How to find desired data from the web using publuc APIs.

MACHINE LEARNING

Study of Supervised and Unsupervised Learning methods. Use of Regression, Classification, Clustering and Dimensionality Reduction algorithms to automize information learning. Study of Neural Networks and Deep Learning methods like MLPs and CNNs.

SIMPLE ELASTIC CLOUD

A 4 months-long project made in collaboration with 2 amazing teamates. This is by far the biggest project I have had in my academic career.

This project consisted in replicating a Cloud System using Virtual Machines as to simulate a client, a ressource manager and a proxy manager for different type of works; and Docker containers for jobs.

Our design succeeded in implementing a partitioned inter-dependent model, Load Balancing, Elasticity and ressource-aware infrastructure. The project was done in Python and used several frameworks such as Flask, Json and Pycurl to help us simulate the cloud system.

The cloud is accompanied with a real-time website accessible localy, showing the current state of the cloud: its nodes and jobs.

AMPLIFY APP WITH TERRAFORM

This is a free AWS Workshop consisting in deploying AWS Amplify web applications using Terraform.

This project succeeds in guiding you step-by-step through the deployment of serverless web applications. One one hand, the app is built using JS and React, offers services such as file storing, log in and resource management. On the other, Terraform files allow for automated deployment of the app using AWS resources.

AWS Amplify allows us to host full-stack apps, with CI-CD pipelines.

Shoutout to Kevon for making this workshop!
